## Prerequisites

### Gradle

#### Windows

```cmd
choco install gradle
```

#### Mac-OS / WSL

```shell
brew install gradle
```

### JDK

#### Mac-OS

```shell
brew install openjdk@17
```

#### WSL - Ubuntu

```shell
sudo apt install openjdk-17-jdk
```
